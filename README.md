## 2. 프로젝트 기능

---
> 예약하기 (좌석)
> 항공편과 예약자 이름, 등급은 client가 지정
> 좌석 번호는 자동으로 지정
> 조회, 생성, 삭제
> 조회 - 
  
### 좌석 예약하기
1. sequence, 금액을 입력 받는다.
2. sequence를 통해 seat를 조회한다.
3. 조회한 seat의 grade에 금액이 맞는지 체크한다.
4. 체크될 경우, username을 등록한다.
